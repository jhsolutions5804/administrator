```

  - name, jumin, address, dept, rank, empType
```

## 4. 급여 계산 공식

### calcAnnualSalary(hourly, empType)

```js
// 현장직 (empType = 'field')
basic      = hourly × 8 × 5 × 4.345
fixedOt    = 0
fixedNight = hourly × 0.5 × 8 × 5 × 4.345
weekly     = hourly × 8 × 4.345
total      = basic + fixedOt + fixedNight + weekly
annual     = total × 12

// 사무직 (empType = 'office')
basic      = hourly × 8 × 5 × 4.345
fixedOt    = hourly × 12 × 4.345
fixedNight = 0
weekly     = hourly × 8 × 4.345
total      = basic + fixedOt + fixedNight + weekly
annual     = total × 12
```

### 공제 계산 (psCalcAll)

```
국민연금     = 지급계 × 4.5%
건강보험     = 지급계 × 3.545%
장기요양보험 = 건강보험 × 12.81%
고용보험     = 지급계 × 0.9%
소득세       = 간이세액표 (부양가족 1인 기준)
지방소득세   = 소득세 × 10%
기숙사       = 직접 입력
```

### 간이세액표 근사치 (calcIncomeTax)

```
≤ 1,060,000원  → 0원
≤ 1,500,000원  → 지급계 × 0.6%
≤ 3,000,000원  → 지급계 × 1.0%
≤ 4,500,000원  → 지급계 × 15% - 420,000
≤ 7,800,000원  → 지급계 × 24% - 825,000
초과           → 지급계 × 35% - 1,680,000
```

## 7. 주의사항

1. **근로계약서 작성이 반드시 먼저** → 연봉계약서에서 시급 연동
2. **연봉계약서 작성이 반드시 먼저** → 급여명세서에서 지급내역 연동
3. **클로저 문제:** `renderLaborForm`, `renderAnnualForm`은 전역 함수로 분리, `window.laborRefreshForm2()` / `window.annualRefreshForm2()`로 새로고침
4. **기숙사 공제:** `oninput` 대신 `onchange` 사용
5. **계좌번호:** 근로자 명부 등록/수정에서 입력 → 급여명세서에서 자동 로드
6. **사원번호:** 자동 부여 (이전 최대값 +1, 3자리 zero-padding), 수기 수정 가능
7. **연봉계약서 기간:** 시작일 선택 시 종료일 자동 1년 설정 (readonly)
8. **급여명세서 지급일:** 기준월 선택 시 익월 말일 자동설정, 수동 수정 가능
